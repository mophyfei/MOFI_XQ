## 🚀 如何匯入 XQ

### 方式 A — 一鍵匯入工具（最簡單）✅

1. [下載 XQ 腳本匯入工具](https://github.com/mophyfei/MOFI_XQ/releases/latest/download/XQ-Script-Importer.exe)
2. 雙擊執行
3. 依指示選擇要匯入的腳本，按確認 ——完成！

### 方式 B — 手動匯入（進階）

**直接貼上**
1. 開啟 XQ 全球贏家
2. 選單：**策略(D) → XScript 編輯器(E)**
3. 點「新增腳本」→ 選對應類別（指標/選股/警示/交易）
4. 貼上 `.xs` 內容，按 **F6 編譯**

**或開啟 .xs 檔**
1. XScript 編輯器 → **檔案 → 開啟 XS 檔案**
2. 選擇下載的 `.xs`，按 **F6 編譯**

編譯成功後依類別啟用：

| 類別 | 啟用方式 |
|------|----------|
| 指標 | 加入指標 → 顯示於技術分析圖 |
| 選股 | 加入選股 → 選股中心使用 |
| 警示 | 加入雷達 → 策略雷達監控 |
| 交易 | 加入自動交易 → 設定自動下單 |
